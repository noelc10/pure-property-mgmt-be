# Pure Property Mgmt. Backend

This is the backend for the Pure Property Mgmt. system, built with Node, HTTP, and TypeScript.

## Project Setup

install the dependencies:

```sh
yarn
```

### Compile for Development

```sh
yarn dev
```

after running the above command, you can use the port shown in the terminal to access the backend server. it should show a message like this after running the said command:

```sh
API running at http://localhost:3000
```

### Front-end

The front-end can be found in the [`Pure Property Mgmt.`](https://github.com/noelc10/pure-property-mgmt) repository. Follow the instructions in that repository to set up and run the front-end application.
